# TF2SR-Custom-Menu-BIK-Collection
A collection of custom menu backgrounds made by the Titanfall 2 Speedrunning community

# Installing custom menu backgrounds
Download the BIK file of the background you want, rename it to "menu_act01" and replace the existing BIK file in "Titanfall2\r2\media" with it.

Keep in mind that custom menus will be replaced with the default one if you verify the game files or reinstall the game.
